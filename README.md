# 🎮 Video Game Sales Data Analysis

![Pandas](https://img.shields.io/badge/Pandas-Data%20Manipulation-150458.svg)

Selamat datang di repositori analisis data penjualan video game! Proyek ini adalah eksplorasi mendalam terhadap dataset penjualan game global untuk mengidentifikasi tren pasar, performa genre, dan pola konsumsi pemain di berbagai wilayah. 

Proyek ini dibuat sebagai bagian dari portofolio **Data Analyst**, menggabungkan ketertarikan pribadi di industri *gaming* dengan kemampuan teknis dalam mengolah data menjadi *insight* bisnis yang *actionable*.

---

## 🎯 Tujuan Proyek
1. Membersihkan dan mempersiapkan dataset penjualan game yang berantakan (*data wrangling*).
2. Mengidentifikasi genre game dan platform dengan penjualan tertinggi secara global.
3. Menganalisis perbedaan preferensi pasar di wilayah utama (NA, EU, JP, Other).
4. Menemukan tren penjualan dari waktu ke waktu untuk memprediksi potensi masa depan.

---

## 🛠️ Tech Stack
- **Bahasa Pemrograman:** Python 3.x
- **Environment:** Google Colab / Jupyter Notebook
- **Data Manipulation:** `pandas`, `numpy`
- **Data Visualization:** `matplotlib`, `seaborn`
- **Dataset:** [Video Game Sales Dataset](https://www.kaggle.com/datasets/gregorut/videogamesales) *(Kaggle)*

---

## 📊 Insight Utama yang Ditemukan
*(Catatan: Sesuaikan poin di bawah ini dengan temuan aktual di notebook-mu)*
- 🏆 **Genre Dominan:** Action dan Sports secara konsisten mendominasi pangsa pasar global, terutama di wilayah Amerika Utara (NA).
- 🌍 **Perbedaan Regional:** Pasar Jepang (JP) menunjukkan preferensi yang sangat kuat terhadap genre RPG, berbeda dengan tren global.
- 📉 **Tren Waktu:** Penjualan game fisik mengalami puncak pada periode 2006-2012 (era PS3/Xbox 360/Wii) sebelum mengalami pergeseran tren.
- 🏢 **Top Publisher:** Nintendo, Electronic Arts, dan Activision menduduki puncak total penjualan kumulatif.

## 🚀 Cara Menjalankan Proyek
1. Clone repositori ini:
   ```bash
   git clone https://github.com/RizalRio/py-data-analyst-game-sales.git
   ```
2. Buka file `Data_Analysis_Penjualan_Game.ipynb` di **Google Colab** atau **Jupyter Notebook** lokal.
3. Pastikan library yang diperlukan sudah terinstal:
   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
4. Jalankan semua sel (*Run All*) untuk melihat proses *data cleaning*, analisis, dan visualisasi.
